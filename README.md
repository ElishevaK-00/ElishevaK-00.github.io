# 💻 Personal Portfolio & Quantitative Trading Dashboard

> 🚀 **Live Production Environment:** [Explore the Live Interface](https://elishevak-00.github.io/)

This repository houses the source code, layout sheets, and asset pipelines for my personal engineering portfolio. The platform is designed to present a clean, high-impact, scannable overview of my academic background, technical toolkits, and production-ready quantitative finance micro-apps.

---

## 🛠️ System Architecture & Front-End Design
- **Host Infrastructure:** Deployed via **GitHub Pages** for rapid, serverless static asset delivery.
- **Layout & Visual Framework:** Built entirely with responsive semantic **HTML5** and custom embedded **CSS3 Flexbox/Grid** mechanics—bypassing heavy third-party frameworks for optimal, lightweight browser paint performance.
- **Dynamic Assets:** Scaled with anchor-point image positioning (`object-position: top;`) to ensure unified multi-device aspect ratios for corporate photography and layout elements.
